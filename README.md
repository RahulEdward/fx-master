# FX-Master 🚀

**Autonomous Forex Trading Engine**

FX-Master is an elite, high-performance algorithmic trading workspace designed for independent quantitative traders, hedge funds, and family offices. It completely abstracts the complexity of broker APIs, WebSocket lifecycles, order deduplication, and retry logic, allowing you to focus purely on algorithm development and alpha generation.

## 🌟 Key Features

- **Ultra-Low Latency Execution:** Direct cross-connects to major liquidity providers ensuring order routing and execution in under 0.1ms.
- **Universal Broker Adapter:** Write your quant strategy once. Our universal adapter normalizes execution across 15+ supported brokers without code changes.
- **Bank-Grade Security:** Military-grade AES-256 encryption. Your API keys are strictly stored in isolated HSM vaults.
- **Advanced Terminal Interface:** A premium, fully responsive "$100M Stealth Startup" dark-mode web dashboard for monitoring live strategies, PnL, and executing trades.
- **Robust Authentication:** Secure registration and login flow featuring real-time, dynamic password strength tracking and UI validation.

## 💻 Tech Stack

- **Frontend Core:** React 18, Vite
- **Styling:** TailwindCSS v3 (Custom Stealth/Pitch-Black UI Theme, Glassmorphism)
- **Routing:** React Router DOM
- **Icons & Graphics:** Custom SVG patterns, animated CSS loaders

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm, yarn, or pnpm

### Local Development Setup 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RahulEdward/fx-master.git
   cd fx-master
   ```

2. **Navigate into the frontend application:**
   ```bash
   cd frontend
   ```

3. **Install all dependencies:**
   ```bash
   npm install
   ```

4. **Spin up the Vite development server:**
   ```bash
   npm run dev
   ```

5. **Initialize your workspace:** 
   Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal output) to access the localized FX-Master Terminal.
